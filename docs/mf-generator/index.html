index

Generate all the possible combinations of molecular formula and calculate for each of them the monoisotopic mass and observed moniisotopic mass (m/z) In the molecular formula there may be a comment after the '$' symbol

index(keys: any, options: object, canonizeMF: boolean, uniqueMFs: boolean, ionizations: string): Array
Parameters
keys (any)
options (object = {})
Name Description
options.limit number (default 10000000) Maximum number of results
options.estimate boolean (default false) estimate the number of MF without filters
canonizeMF (boolean = true) Canonize molecular formula
uniqueMFs (boolean = true) Force canonization and make MF unique
ionizations (string = '') Comma separated list of ionizations (to charge the molecule)
Returns
Array:
