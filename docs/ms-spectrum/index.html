getBestPeaks

Filter the array by taking the higher peaks and only keep one per slot. There are 2 different slots, the smallest one will have the extra annotation close to true

getBestPeaks(peaks: array, options: object, from: number?, to: number?, limit: number, threshold: number, numberSlots: number, numberCloseSlots: number): array
Parameters
peaks (array) array of all the peaks
options (object = {})
from (number?) min X value of the window to consider
to (number?) max X value of the window to consider
limit (number = 20) max number of peaks
threshold (number = 0.01) minimal intensity compare to base peak
numberSlots (number = 10) define the number of slots and indirectly the slot width
numberCloseSlots (number = 50)
Returns
array: copy of peaks with 'close' annotation

options.limit number (default undefined) maximal number of peaks (based on intensity)
Returns

isContinuous

When a spectrum is continous ?

  • has more than 100 points
  • deltaX change can not be more than a factor 2
  • deltaX may not be larger than 0.1
  • if y is zero it does not count
isContinuous(spectrum: any)
Parameters
spectrum (any)
